# Controlled device measurements

Place only files generated from a physical device here.

Synthetic fixtures must never be renamed or promoted into this directory.

Expected naming example:

```text
pixel6a_session_001.json
```

Each file must include:

```json
"evidence_level": "controlled_device_measurement"
```

Until a Pixel 6a / laptop controlled capture is executed, this directory intentionally
contains no measurement JSON.
