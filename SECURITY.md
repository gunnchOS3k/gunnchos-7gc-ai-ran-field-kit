# Security Policy

## Supported versions

Main / default branch only for active portfolio hardening.

## Reporting

Open a private security advisory on GitHub or contact the maintainer.

## Scope

- No collection of private payload traffic
- No unauthorized RF transmission in any contribution
- Opt-in telemetry only; document schemas in PRs
- Youth safety: see `docs/PRIVACY_AND_YOUTH_SAFETY.md` or `docs/PRIVACY_AND_ETHICS.md`
